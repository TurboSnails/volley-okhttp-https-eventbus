# OkVolley
------
此框架封装了Volley OkHtp Https 还有可以自动实现网络访问的dialog显示

Studio引入方式：
 >   compile 'com.turbo.okvolley:okvolley:1.0.0'


**此项目目前停止维护，此项目的升级版:**
### [TurVolley](https://github.com/yuanyang5917/TurVolleyDemo)

 >1. 支持https
 >2. 支持缓存
 >3. 支持自动显示 隐藏dialog
 >4. 支持防止重复点击
